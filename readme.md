# Nintrea Webiste 🌈🌞

Compiled with various documents to support the study of students or those who like to research documents on the Internet.

<br>

## To Start Project ♻️

```sh
npm install
```

#### Develop Project 🚀

```sh
npm run dev
```

#### Production 🚀🌈♻️
```sh
npm run build
```

#### Start Production 🚀🌈♻️
```sh
npm run start

# Or

npm start
```
